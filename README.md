Asendia Integration
===================

Provides integration with Asendia.


Requirements
------------

 * PHP 5.4+


Installation
------------

```bash
$> composer require cpap/asendia
```

While in early development, you may be required to be a little more specific:

```bash
$> composer require cpap/asendia:^0.0@dev
```
